# 0. 파이썬 웹 프레임워크 기초 (Flask - 설치 및 기본)

## 1. Flask (플라스크) 개요

Flask는 파이썬으로 구동되는 초간단 마이크로 웹 프레임워크이다. 최소한의 기본 기능만 제공하며, 필요한 기능은 나중에 확장 모듈을 통해 자유롭게 붙여서 사용할 수 있도록 설계되었다.

과거에 파이썬 기본 모듈로 사용하던 `CGIHTTPServer`는 기능이 너무 약하고 한계가 뚜렷하기 때문에, 본격적인 웹 개발을 위해 Flask를 사용한다. 단, Flask에 내장된 개발용 서버만으로는 대규모 트래픽을 감당하기 부족하므로 실제 상용 웹 서비스를 배포할 때는 **uWSGI**나 **Gunicorn** 같은 전문적인 WSGI(Web Server Gateway Interface) 서버를 연동하여 사용해야 한다. (Flask 자체 내장 서버는 주로 교육용이나 개발 테스트용으로 쓰인다.)

### 주요 활용 분야
웹사이트 구축, 백엔드 API 서버, 챗봇 서버, IoT 제어 서버, 학습용 서버, 데이터 시각화 대시보드(Dashboard) 등 다양한 분야에 널리 활용된다.

### Flask의 장점
* **배우기 쉽다**: 코드가 매우 직관적이고 구조가 단순하다.
* **가볍다**: 불필요한 기본 모듈이 억지로 포함되어 있지 않아 가볍다.
* **확장하기 쉽다**: 필요한 라이브러리(DB 연동, 폼 검증 등)만 골라서 유연하게 확장할 수 있다.


## 2. Flask 설치
파이썬 환경이 구성되어 있다면, 터미널(또는 CMD)에서 `pip` 명령어를 통해 매우 간단하게 설치할 수 있다.
```bash
pip install flask
```


## 3. Flask 기본 웹 서버 구동 실습
설치가 완료된 후, 가장 기본이 되는 Flask 웹 서버를 띄우는 뼈대 코드이다.

```python
from flask import Flask        # Flask 클래스 불러오기 (웹 서버 생성에 필요)

app = Flask(__name__)          # Flask 앱(웹 서버) 객체 생성. __name__은 현재 모듈 이름을 의미한다.

@app.route('/')                # 웹 브라우저에서 '/' (기본 루트) 경로로 요청이 오면 아래 함수를 실행한다. (라우팅)
def hello_world():             # 라우트('/') 요청을 처리할 함수 정의
    return 'Hello, World!'     # 브라우저 화면에 출력할 문자열을 반환한다.

if __name__ == '__main__':     # 이 파이썬 파일을 직접 실행했을 때만 아래 코드를 동작시킨다.
    # app.run()                # Flask 기본 개발 서버 실행 (기본 포트: 5000, 로컬 전용)
    
    app.run(
        debug=True,            # 디버그 모드 ON
        host='0.0.0.0',        # 외부 환경에서도 접속 가능하도록 IP 허용
        port=5001              # 기본 포트인 5000 대신 5001 포트를 사용
    )
```

### 주요 실행 옵션 설명
* **`@app.route('경로')`**: 사용자가 브라우저 주소창에 특정 URL 경로를 입력했을 때, 어떤 파이썬 함수를 매칭하여 실행할지 연결해 주는 **라우팅(Routing)** 역할을 한다.
* **`debug=True`**: 디버그 모드를 켠다. 코드를 수정하고 저장하면 서버가 자동으로 변경 사항을 감지하여 재시작(Reload)되므로 개발 시 매우 편리하다. (`debug=False`이거나 옵션이 없으면 중간에 코드를 고쳐도 서버를 수동으로 껐다 켜기 전까지는 반영되지 않는다.)
* **`host='0.0.0.0'`**: 로컬호스트(`127.0.0.1`)뿐만 아니라, 같은 네트워크(공유기)에 연결된 다른 기기(스마트폰 등)에서도 해당 컴퓨터의 IP 주소를 입력해 들어올 수 있도록 외부 접속을 허용하는 옵션이다.
